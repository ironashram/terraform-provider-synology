# Container: Project (Resource)

A Docker Compose project for the Container Manager Synology API.

## Example Usage

```terraform
resource "synology_container_project" "foo" {
  name = "foo"

  service {
    name = "bar"

    image {
      name = "nginx"
      tag  = "latest"
    }

    port {
      container_port = 80
      host_port      = 80
    }

    logging {
      driver = "json-file"
    }
  }
}
```
